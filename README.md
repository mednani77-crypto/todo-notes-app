# ملاحظاتي

تطبيق ملاحظات عربي أولاً، متجاوب وقابل للتثبيت، مبني بـ React وVite مع نسخ صوتي مجاني يعمل محلياً في المتصفح.

## الوظائف المنفذة

- محرر Tiptap غني يدعم RTL/LTR، العناوين، التنسيق، الروابط، القوائم وقوائم المهام.
- حفظ تلقائي مع حالة واضحة ومنع فقدان التعديلات عند الإغلاق.
- بحث فوري في العناوين والمحتوى والوسوم والمجلدات، مع الفرز والتصفية.
- تثبيت، مفضلة، أرشيف، سلة محذوفات، استعادة وحذف نهائي مؤكد.
- مجلدات ووسوم وألوان ومرفقات صور/ملفات مخزنة في IndexedDB.
- تسجيل صوتي مع إيقاف مؤقت واستئناف ومعاينة وإعادة محاولة النسخ.
- نسخ صوتي حقيقي داخل الجهاز عبر Whisper وTransformers.js، دون مفتاح API أو رفع التسجيل.
- تصدير PDF مباشر مع العربية والنص المختلط، A4/Letter، اتجاه الصفحة، التواريخ والمرفقات.
- تصدير نص وMarkdown ونسخة JSON، واستيراد آمن بالدمج أو الاستبدال المؤكد.
- سمات فاتحة وداكنة وتلقائية، تنقل لوحة المفاتيح، حركة مخفّضة وأهداف لمس مناسبة.
- Manifest وService Worker لقشرة التطبيق والعمل دون اتصال للموارد المحمّلة سابقاً.

## البنية

- الواجهة: React 19، Vite، Tiptap، Lucide، DOMPurify.
- النسخ الصوتي: نموذج Whisper متعدد اللغات بصيغة ONNX يعمل داخل Web Worker ويُخزّن في Cache API.
- التخزين: `localStorage` للملاحظات والإعدادات، وIndexedDB لبيانات المرفقات.
- المصادقة: لا توجد حسابات أو مزامنة خادم في النسخة الحالية؛ البيانات محلية لكل متصفح.
- النشر: موقع ثابت على Render يقدم ملفات `dist` عبر CDN؛ لا توجد خدمة خلفية مطلوبة.

## الحفاظ على البيانات والترحيل

يبقى مفتاح الملاحظات القديم كما هو: `todo-notes-app:v1`. عند أول تشغيل:

1. تُقرأ السجلات القديمة `{ id, text, completed, createdAt, updatedAt }`.
2. تُنشأ نسخة احتياطية خام قبل الترحيل.
3. تُطبّع الملاحظات إلى المخطط رقم 3 مع إبقاء حقل `text` للتوافق.
4. يُحفظ رقم المخطط لمنع تكرار الترحيل.
5. إذا كانت البيانات تالفة، تُحفظ في مساحة الاسترداد ولا تُستبدل بصمت.

لا توجد قاعدة بيانات إنتاجية أو تغييرات مخطط مدمرة.

## التشغيل المحلي

```bash
pnpm install --frozen-lockfile
pnpm dev
```

لمعاينة نسخة الإنتاج محلياً:

```bash
pnpm build
pnpm preview
```

## إعداد النسخ الصوتي

تم اختيار `onnx-community/whisper-tiny` مع `@huggingface/transformers` لأنه مجاني، متعدد اللغات، ولا يرسل التسجيل إلى أي خادم. يفك المتصفح الصوت ويحوّله إلى mono 16 kHz، ثم يشغّل Whisper في Web Worker حتى تبقى الواجهة مستجيبة.

- لا تحتاج إلى مفتاح OpenAI أو اشتراك مدفوع.
- في أول استخدام فقط، ينزّل المتصفح ملفات النموذج الكمي من Hugging Face (نحو 45–50 MB حسب المتصفح) ويحفظها في Cache API.
- بعد اكتمال التنزيل يمكن إعادة استخدام النموذج من الذاكرة المؤقتة؛ إذا مسح المستخدم بيانات الموقع فسيلزم تنزيله مجدداً.
- التسجيل نفسه يبقى على الجهاز، ويظل محفوظاً إذا فشل التنزيل أو النسخ ويمكن إعادة المحاولة.
- العربية والإنجليزية مدعومتان. يوفر Whisper أيضاً الصومالية والأمهرية، لكن دقتهما في النموذج الصغير تتفاوت بحسب اللهجة وجودة الصوت.
- الأجهزة الضعيفة قد تستغرق وقتاً أطول لأن الاستدلال يجري على المعالج محلياً.

## PDF

يستخدم التصدير `html2pdf.js` لإنشاء ملف قابل للتنزيل مباشرة، لا مجرد أمر طباعة. يُرسم DOM بعد اكتمال الخطوط والصور للمحافظة على تشكيل العربية وRTL. تتوفر معاينة طباعة منفصلة كخيار ثانوي.

## الفحوص

```bash
pnpm lint
pnpm test
pnpm build
```

تغطي الاختبارات الترحيل، البحث والفرز، حالات الملاحظة، المجلدات، السمات، النسخ الصوتي الناجح والفاشل وإعادة المحاولة، وتهيئة PDF العربية والمختلطة.

## النشر على Render

يوفر `render.yaml` موقعاً ثابتاً واحداً. في Render:

1. اربط المستودع أو طبّق الـ Blueprint.
2. تحقق من أن Runtime هو `Static` وأن Build Command هو `corepack enable && pnpm install --frozen-lockfile && pnpm build`.
3. اضبط Publish Directory على `dist` وأضف Rewrite من `/*` إلى `/index.html`.
4. لا توجد متغيرات بيئة أو مفاتيح سرية مطلوبة.

## حدود معروفة

- لا توجد مصادقة أو مزامنة بين الأجهزة؛ مسح بيانات المتصفح يزيل البيانات المحلية ما لم تُصدّر نسخة JSON.
- المرفقات محلية ولا تُزامن إلى Render.
- الإشعارات الخلفية والتذكيرات غير مدعومة.
- يلزم اتصال في أول استخدام للنسخ لتنزيل النموذج؛ سرعة النسخ ودقته أقل من خدمات السحابة الكبيرة على بعض الأجهزة.
- قشرة PWA تعمل دون اتصال للموارد المخبأة؛ لا توجد استراتيجية مزامنة تضارب لأنها لا توجد مزامنة خادم.
